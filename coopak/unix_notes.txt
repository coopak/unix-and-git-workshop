rm : deletes file
mv : moves files (also used to rename)
cd : changes directories
cp : copies file
ls : lists all files/folder in current dir
mkdir : creats new fold in current dir
rm -rf : removes a folder and all subfolders
chmod : used to change permissions on files
sudo : execute line as root
sudo su : enable root env going forward
pwd : show you which directory you're in
nano : simple text editor
more : gives you sample of file in terminal
grep : looks for text in specified files or folders
